# 🎨 Петербург сквозь года — Лендинг курса акварели

> Адаптивный лендинг для онлайн-курса по акварельной живописи Санкт-Петербурга под руководством художника Владимира Заруцкого. Проект демонстрирует навыки современной веб-верстки и адаптивного дизайна.

---

## ✨ Ключевые особенности

### 📱 Адаптивный дизайн
- Полная адаптивность под мобильные устройства, планшеты и десктоп
- Использование Bootstrap 4.6 сетки для гибкой верстки
- Кастомные медиа-запросы для точечной настройки

### ⚡ Интерактивные элементы
- **Аккордеон FAQ** — кнопки меняют цвет с синего на оранжевый при открытии
- **Карточки тарифов** — переключение между рублями и долларами
- **Раскрывающийся список** материалов курса
- **Плавная навигация** — якорные ссылки с анимацией прокрутки

### 🎨 Дизайн и визуал
- Кастомная цветовая палитра
- Уникальный шрифт Bernier Distressed для заголовков
- Текстурированные фоны для разных секций
- Эффекты наведения на всех интерактивных элементах
---

## 🛠️ Технологический стек
- **HTML5** — семантическая разметка
- **CSS3** — кастомные стили, Flexbox, анимации
- **JavaScript (ES6+)** — логика интерактивных элементов
- **jQuery 3.7.1** — упрощение работы с DOM
- **Bootstrap 4.6.2** — адаптивная сетка и компоненты
- **Google Fonts** — шрифты PT Sans и Bernier Distressed

---

### JavaScript/jQuery
- Обработка событий клика
- Анимации (`slideToggle`, `fadeIn`, `animate`)
- Работа с классами и атрибутами DOM
- Переключение состояний (активная/неактивная кнопка)

### CSS и дизайн
- Кастомные CSS-классы (`.orange-title`, `.color-blue`, `.orange-list`)
- Псевдоэлементы для создания маркеров списка
- Градиенты для кнопок
- Эффекты трансформации и перехода
